# Data Sources

This repo does not bundle a complete national data center registry.

Recommended source lanes:

- **PNNL / IM3 Open Source Data Center Atlas** for public atlas-style facility records.
- **Epoch AI Frontier Data Centers** for large AI/frontier data center records and timelines.
- **Public permits** for local status, location, and construction claims.
- **Public utility filings** for grid, power, and interconnection claims.
- **Public operator announcements** for owner/operator claims.
- **Public news or government records** for corroboration and dispute notes.

For every source, preserve:

- source name
- source category
- retrieved date
- claim text
- confidence level
- URL or citation when appropriate
