# Public Data Boundary

DataCenterLedger Explorer is designed for civic transparency and public-record review.

## Allowed

- Public datasets
- Public permits
- Public utility filings
- Public company announcements
- Public news and government records
- City, county, or state-level location precision when higher precision is not needed
- Receipt-backed claims with source names, dates, confidence, and review status

## Not allowed

- Non-public records
- Access instructions
- Operational details that are not clearly public
- Data that violates source terms
- Treating unverified claims as final facts

## Location precision rule

Use the least precision needed for the review purpose. Preserve the source trail for every location claim.
